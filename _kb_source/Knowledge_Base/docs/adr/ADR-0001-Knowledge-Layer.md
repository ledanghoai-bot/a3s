# ADR-0001 --- Knowledge is Source Code

## Status

Accepted

## Decision

Knowledge Base is managed exactly like source code.

-   Git version control
-   Pull Request review
-   Versioning
-   Rollback support
-   No direct production edits

## Rationale

Knowledge is a core asset of Alpha3S and Beta3S. Treating it as source
code improves quality, traceability and long-term maintainability.
